The journey.

Five stages, bottom to top. Each one is a section of the guide — and the concepts behind them live in the docs below.

1

Foundations

The networking you must own: private vs public IPs, NAT, CGNAT, ports, DNS, and whether your connection can even host a server.

2

The hypervisor

What virtualization actually is and how it works — then install Proxmox as the bare-metal base and run your OS as a VM inside it.

3

Build the base

A Linux VM, SSH, a fixed internal IP, the daily commands, and Docker + Compose as the way you run everything.

4

Deploy your apps

A reverse proxy, your domain on Cloudflare, DNS records, and getting to the internet safely — Cloudflare Tunnel or port forwarding.

5

Run it like a pro

SSH keys, a firewall, fail2ban, backups and snapshots, and the maintenance habits that keep it alive and secure.
